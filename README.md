# Superconductor Data Analysis (MATLAB)

MATLAB script developed during my MSc in Physics to analyze experimental measurements of superconducting materials.

## Features

- Automatic extraction of experimental data
- Data cleaning and preprocessing
- Column conversions
- Automatic generation of plots
- Export of processed data to Excel
- Calculation of statistical parameters

## Data format

The script reads experimental files with extensions:

.Cn1  
.Cn2  

These files must be located in the same folder as the MATLAB script.
